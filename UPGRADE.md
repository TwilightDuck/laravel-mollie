# Migrating from Laravel-Mollie v2.x to v3

### Step 1: Update composer dependencies

Update `composer.json` to match this:

```json
"require": {
    "mollie/laravel-mollie": "^3.0"
}
```

Then run `composer update mollie/laravel-mollie`.
